## 说明 {#说明}

通过 cpio 命令解压 ramddisk 解压系统后重新打包制作镜像。


## 步骤 {#步骤}

```bash
# 查看文件格式
➜ file initrd.img
initrd.img: XZ compressed data, checksum CRC32

# 修改文件名，并解压
➜ mv initrd.img initrd.img.xz
➜ xz -d -k initrd.img.xz

# cpio解压系统
➜ mkdir rootfs && cd rootfs
➜ cpio -ivd < ../initrd.img

# 修改之后重新打包、压缩（rootfs/)
➜ find . | cpio -o -H newc > ../initrd.img.cpio
➜ cd ..
➜ xz -z -k initrd.img.cpio
➜ ls
initrd.img.cpio.xz
➜ mv initrd.img.cpio initrd.img
```
