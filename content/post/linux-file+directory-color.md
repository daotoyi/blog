-   白色：表示普通文件
-   蓝色：表示目录
-   绿色：表示可执行文件
-   红色：表示压缩文件
-   浅蓝色：链接文件
-   红色闪烁：表示链接的文件有问题
-   黄色：表示设备文件
-   灰色：表示其它文件
